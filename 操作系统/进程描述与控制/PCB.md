# PCB控制块

​        在操作系统中，我们重要的就是进程，但是我们如何来对进程进行控制呢？这里我们就要说到重要的一个存储结构：PCB控制块。这个是为了系统的描述和管理进程。

## PCB的作用

1. 作为独立运行的基本单位的标识
2. 实现间断运行的方式
3. 提供进程管理所需的信息
4. 提供进程调度所需的信息

## PCB中的信息

1. 进程标识符号

   * 外部标识符号

     作用是给用户查看的

   * 内部标识符号

     为了计算机内部的一个标识

2. 处理机状态

   * 通用寄存器
   * 指令计数器
   * 程序状态字寄存器
   * 用户栈寄存器

3. 进程调度的信息

   包含的信息：

   * 进程状态
   * 进程优先级
   * 进程调度所需的其他信息
   * 事件

4. 进程控制信息

   * 程序和数据的地址
   * 进程同步和通信的机制
   * 资源清单
   * 连接指针

## 进程的组织方式

1. 线性方式
2. 链接方式
3. 索引方式
